# Projeto-3_Site-de-uma-editora-de-livros
Projeto consiste em um site simples de uma editora de livros. O projeto foi desenvolvido durante o curso de HTML e CSS.
Diferente dos outros projetos, neste foi usado o "reset.css" que serve para remover estilos padrão aplicados automaticamente
pelos navegadores, e garantir a consistência visual entre diferentes navegadores.
